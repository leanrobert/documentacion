# FAQ para Usuarios

## Preguntas Generales

### ¿Cómo accedo al sistema?

Puedes acceder al sistema a través de la página principal. La documentación de usuarios es pública y no requiere autenticación.

### ¿Dónde encuentro la información que necesito?

La documentación está organizada en secciones temáticas en el menú lateral izquierdo.

### ¿Quién puede ayudarme si tengo problemas?

Contacta al administrador del sistema para obtener ayuda adicional.

## Funcionalidades

### ¿Qué puedo hacer como usuario?

Como usuario puedes:

- Consultar toda la documentación pública
- Acceder a tutoriales y guías
- Revisar las FAQ
- Buscar información específica

### ¿Necesito una cuenta para usar el sistema?

No, la documentación de usuarios es completamente pública y no requiere registro.
